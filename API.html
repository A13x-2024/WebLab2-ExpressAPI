API Documentation

Endpoints

Get / Post

Endpoint: GET /potatisar

Description: Hämtar alla potatisar.

Endpoint: POST /potatis

Description: Lägger till potatis.

Example Request:

fetch('https://labb2-web-gpgmftgjahawhmg3.westeurope-01.azurewebsites.net/potatisar/1')
    .then(response => response.json())
    .then(data => console.log(data));

Example Response:

{
    "id": 1,
    "Name": "Kakashi-Potato",
    "Type": "Legendary",
    "Rank": "9001" (Over 9000)
}
